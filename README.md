# DashCrafters Landing Page

A modern, professional landing page for a data analytics and Power BI reporting services website. Built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Modern Design**: Clean, tech-inspired design with cool-toned color palette
- **Responsive**: Fully responsive design for desktop, tablet, and mobile
- **Animations**: Smooth animations and interactions using Framer Motion
- **Dark Mode**: Built-in dark/light mode toggle
- **SEO Optimized**: Comprehensive SEO meta tags and structured data
- **Performance**: Optimized for Core Web Vitals and fast loading
- **Accessibility**: WCAG compliant with proper contrast and alt text

## Sections

1. **Hero Section**: Animated dashboard motif with compelling headline and CTA
2. **Services**: Interactive cards showcasing core services with animations
3. **Portfolio**: Carousel and grid view of project showcases
4. **About**: Professional introduction with expertise and values
5. **Contact**: Comprehensive contact form with validation
6. **Footer**: Complete footer with links, social media, and newsletter signup

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Forms**: React Hook Form with Zod validation
- **Icons**: Lucide React
- **Deployment**: Optimized for Vercel

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd dashcrafters-landing
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Customization

### Colors
The color scheme can be customized in `tailwind.config.js`:
- Primary: Blue tones
- Secondary: Purple tones
- Accent colors for different sections

### Content
Update content in the respective component files:
- `src/components/Hero.tsx` - Hero section content
- `src/components/Services.tsx` - Services and features
- `src/components/Portfolio.tsx` - Portfolio items
- `src/components/About.tsx` - About section content
- `src/components/Contact.tsx` - Contact information

### Images
Replace placeholder images in:
- Portfolio items (currently using Unsplash)
- Hero background elements
- About section graphics
